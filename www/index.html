    Reset Parameters

    Use with reset command prefix, e.g. r prey=40.

    prey=40
    Number of prey
    pred=6
    Number of predators
    food=60
    Number of foods
    prey-n=9
    Prey hidden neurons
    pred-n=9
    Predator hidden neurons
    prey-p=9
    Prey photoreceptors
    pred-p=9
    Predator photoreceptors
    prey-fov=225
    Prey FOV angle (degrees)
    pred-fov=225
    Predator FOV angle (degrees)
    prey-speed=1.0
    Prey speed multiplier
    pred-speed=1.0
    Predator speed multiplier

    Examples:

    reset prey=100 pred=10 food=100
    r prey=100 pred=10 food=100 prey-n=12 pred-n=8 prey-p=6 pred-p=4
    r prey-fov=200 pred-fov=240
    r prey-speed=1.15 pred-speed=0.9
